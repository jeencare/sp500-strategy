# S&P 500 定投及增益策略

这是一个基于 S&P 500 指数的定投与抄底策略回测工具，可生成可视化 HTML 报告。

## 功能特点
- 实时更新数据（yfinance）
- 支持多种 ETF（SPY, VOO, IVV, SSO, UPRO）
- 两种策略模式：定投 / 抄底
- 交互式图表与交易流水

## 如何运行
1. 安装依赖：`pip install -r requirements.txt`
2. 运行脚本：`python sp500_strategy.py`
3. 脚本会自动下载数据并生成 HTML 报告，并使用浏览器打开。
